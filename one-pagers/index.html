Foundations
01Architecture

Lakehouse & Medallion

One platform fusing a lake's cheap open storage with a warehouse's reliability — refined Bronze → Silver → Gold.

 02Governance

Unity Catalog

The unified governance layer — one permission model, one lineage graph, one audit trail across every workspace.

 03Storage Format

Delta Lake

Parquet wrapped in a transaction log — real ACID, time travel, and schema control on cloud object storage.

 04Ingestion

Auto Loader

A streaming source that ingests new files as they land — no manual bookkeeping of what's already processed.
Pipelines & Orchestration
05Data Engineering

Lakeflow Declarative Pipelines

Define the tables you want; the framework builds the DAG, the compute, and the incremental updates. (formerly DLT)

 06Orchestration

Lakeflow Jobs

The native lakehouse scheduler — wire tasks into a DAG, trigger on a clock or new data. (formerly Workflows)

 10Streaming

Structured Streaming

Run the same DataFrame code over an unbounded stream — exactly-once results via checkpoints.
Spark, Performance & SQL
07Apache Spark

Spark Architecture

Code → parallel work: a driver plans, executors run, and nothing happens until an action fires.

 08Performance

Spark Performance Tuning

Let AQE re-plan at runtime, pick the right join, and stop reading data you don't need.

 09Delta Optimization

Delta Optimization & Liquid Clustering

Compaction, data skipping, and the clustering model that replaces partitioning + Z-order with no rewrite.

 11SQL & BI

Databricks SQL & Photon

Serverless SQL warehouses on Photon — the vectorized C++ engine that accelerates work with no code change.

 12Compute

Compute & Cluster Types

All-purpose, jobs, SQL warehouses, serverless — plus access modes, policies, autoscaling, spot, and Photon.
AI, ML, GenAI & Agents
13AI / ML

Model Serving + MLflow

Track and register with MLflow, govern in Unity Catalog, deploy behind one autoscaling REST endpoint.

 14AI / ML

Mosaic AI Vector Search

Serverless similarity search that turns Delta tables into governed vector indexes — the retrieval layer for RAG.

 15AI / BI

AI/BI Genie & Dashboards

Ask governed data questions in plain English — Genie writes and runs the SQL; dashboards share the result.

 19GenAI / Agents

Agent Bricks

Declaratively build domain agents on your data — describe the task, it generates the eval and tunes the agent.

 20GenAI / Agents

Mosaic AI Agent Framework

The code-first way to author, log, evaluate, deploy, and trace production agents — built around MLflow.

 21GenAI / Eval

GenAI Evaluation

Measure GenAI/agent quality with LLM-as-judge scorers, eval datasets, human labeling, and production monitoring.

 22MLOps

LLMOps on Databricks

MLOps extended for GenAI: prompts, RAG, and agents through one develop → evaluate → deploy → monitor loop.

 23MLOps

MLOps on Databricks

Train, register, deploy, and monitor classic predictive models — DevOps + DataOps + ModelOps under one UC plane.
Sharing, OLTP, DevOps & Apps
16Data Sharing

Delta Sharing & Marketplace

An open protocol to share live data and AI assets across orgs and clouds — no copying, no lock-in.

 17OLTP

Lakebase Postgres

A managed Postgres-compatible OLTP database built into Databricks — low-latency txns next to your lakehouse.

 18DevOps / CI-CD

Databricks Asset Bundles

Infrastructure-as-code for the lakehouse — describe resources in YAML and ship them dev → prod with one CLI.

 24Databricks Apps

Apps Authorization

Act as the app's service principal (shared data) or as the signed-in user via OBO (per-user data) — plus forwarded identity headers.
