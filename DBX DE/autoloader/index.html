How to use the notebook: in Databricks, go to Workspace → (kebab menu) → Import → File and pick auto_loader_demo.py — the # COMMAND ---------- markers turn into cells. Edit CATALOG / SCHEMA in the Config cell to one you own (needs CREATE rights). The two HTML pages open directly in any browser — no setup.
